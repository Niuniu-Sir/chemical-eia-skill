- Canonical Skill for technician-guided extraction, provisional proposals, traceable decisions, and iterative review.

- Reproducible wheel, sdist, Skill ZIP, and `SHA256SUMS.txt` release assets with exact member inspection.

### Security

- Strict public file boundaries and layered source, index, history, object, and archive scanning.
