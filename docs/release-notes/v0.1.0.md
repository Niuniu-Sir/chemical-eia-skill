# v0.1.0 Preview release notes

## Install and inspect

The release contains a universal wheel, a source distribution (sdist), a Canonical Skill ZIP, and `SHA256SUMS.txt`. Download all four files into one directory, verify the checksums before installation, and inspect the archive member lists if your environment requires supply-chain review.

Install the wheel with `python -m pip install chemical_eia_core-0.1.0-py3-none-any.whl`. The sdist can be built offline after build dependencies have been prepared in the isolated build environment. Extract the Skill ZIP into the Codex skills directory while preserving its top-level directory.

Verify `SHA256SUMS.txt` with a SHA-256 tool available on your operating system or with the repository verification helper. A mismatch, missing file, unknown file, duplicate entry, or altered manifest must stop installation.

## What is included

- Deterministic model loading, validation, balance diagnostics, and rendering.
- Canonical Skill and Claude Code adapter boundary.
- Completely fictional minimal example and public tests.
- Exact release-asset contracts and layered sensitive-content checks.

## Known limitations

This Preview does not extract arbitrary office documents by itself, does not invent missing reaction definitions, does not resolve enterprise conflicts without review, and does not provide a regulatory conclusion. YAML input depends on an optional parser being available; JSON remains the dependency-free interchange format.

## Technician responsibility

AI-origin candidates remain preliminary. A formal result requires an explicit technician decision, preserved provenance, successful deterministic validation, and any necessary enterprise confirmation. Installing or running the software does not transfer that responsibility to the program.
